easycsv is a csv builder class that, as the name states, makes object-to-csv easy.

### how to use

```java
Foo foo = new Foo();
foo.setId(1);
foo.setName("bar");

EasyCSV builder = new EasyCSV(foo);

String csv = builder.build()
```

### output

```
Id,Name
1,bar
```
